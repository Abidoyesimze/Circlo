# Circlo

Trustless rotating savings circles (ajo / esusu / tanda / chama / susu) on Stellar,
enforced end-to-end by a Soroban smart contract instead of a human organizer.

> Full documentation is being written alongside the implementation — see
> [`contracts/circlo`](contracts/circlo) for the smart contract and
> [`frontend`](frontend) for the app.
