# Circlo frontend

Vite + React 18 + TypeScript + Tailwind CSS + shadcn-style UI primitives.

```bash
npm install       # from the repo root (npm workspaces)
npm run dev        # start the dev server
npm run build       # typecheck + production build
npm run lint       # oxlint
npm test           # vitest
```

See the [root README](../README.md) for the full project overview, contract
architecture, and testnet deployment details.
